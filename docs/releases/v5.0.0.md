# Proveria V5.0.0

V5 is the product-led workflow and enterprise administration release.

## Highlights

- Drag-and-drop producer file submission.
- Clearer multi-file submission progress and automatic detail reveal.
- Project/file template removal from user-facing creation flows.
- Searchable, sortable, filterable, and paginated verifier access grants.
- Searchable, sortable, filterable, and paginated verifier access requests.
- Approval and denial reason handling for verifier access requests.
- Explicit workspace switching.
- Workspace membership and access controls.
- Admin Events table polish with search, filters, sorting, paging, expandable
  event detail, and refresh.
- Admin event/log export for workspace-scoped workflows.
- Event/log export filters for project, actor, category, and date range.
- Evidence export manifest and durable export job foundation.
- Recent evidence export job visibility in desktop.
- Saved evidence export job manifest retrieval from desktop, CLI, API, and SDK
  surfaces.
- Downloadable JSON evidence artifact bundles for evidence export jobs.
- Organization-scoped evidence export manifests and bundle jobs for
  organization admins.
- Evidence export job lifecycle metadata for progress, retries, expiration, and
  retention policy.
- Explicit cleanup for expired evidence export bundle objects with retention
  deletion audit events.
- CLI inspect and unpack support for reviewing JSON evidence bundles and
  turning them back into manifest and artifact files.
- CLI collect workflow for creating, downloading, inspecting, and unpacking an
  evidence package into one local directory.
- CLI ZIP and tar archive packaging for evidence bundles and collected evidence
  packages.
- CLI evidence package checks for standalone bundle JSON files and collected
  evidence directories.
- Generic OIDC authentication for configured providers.
- Microsoft Entra ID/Azure AD local OIDC setup documentation.

## Deferred

- Additional OIDC provider administration UI.
- Google sign-in, Google external identity connection, Google Drive local
  import, Google Drive Picker-based browsing, and automatic Drive download.
- Automatic scheduling for expired evidence export bundle cleanup.
- Broader V5 webhook coverage.
