# Proveria V4.0.0

V4 closes as the internal commercial API foundation release.

## Highlights

- Tenant-scoped API keys.
- Public V1 API foundation.
- Stable public API response/error envelopes.
- OpenAPI document.
- API-created hash attestations.
- API verifier access grant and revoke.
- Signed webhook delivery foundation.
- CLI alpha.
- TypeScript SDK alpha.

## Deferred

- Python SDK spike remains paused.
- Public developer repositories and package publishing remain paused.
- Lookup-result webhook events remain deferred.
- Further developer-surface expansion moves behind V5 product workflow and
  enterprise administration decisions.

## Next

V5 becomes the active planning target: product-led workflow polish, workspace
management, Google authentication/Drive, admin log export, and evidence/artifact
exports.
