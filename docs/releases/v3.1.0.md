# Proveria V3.1.0 Release Notes

Release date: May 25, 2026

## Summary

V3.1.0 is the V3 hardening release. It does not add new product scope beyond
V3.0.0; it locks in automated confidence for the OCR, native text, and exact
image proof paths that passed human QA.

## What Changed

- Added focused scanned-PDF OCR smoke coverage with `pnpm smoke:ocr-pdf`.
- Added focused exact-image smoke coverage with `pnpm smoke:exact-image`.
- Kept focused native PDF text content-proof smoke coverage with
  `pnpm smoke:pdf-text-layer`.
- Updated local QA docs and the V3 completion checklist to include the complete
  focused smoke set.

## Validation

Validated on fresh `main`:

- `pnpm smoke:pdf-text-layer`
- `pnpm smoke:exact-image`
- `pnpm smoke:ocr-pdf`
