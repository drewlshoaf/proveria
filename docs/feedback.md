# Product Feedback Notes

Use this file for short-lived product feedback that should either become a
tracked backlog item or be marked resolved after implementation.

## Attestation Submission UX

Status: resolved in V5 submission flow improvements.

- The submit action was too low on the page and easy to miss.
- After submitting an attestation, the page felt idle while work continued.
- The flow needed visible progress during hashing, submission, validation, and
  confirmation.
- When an attestation became ready, its detail view should open automatically.
- Returning to New Attestation or navigating back should clear submitted form
  state.
